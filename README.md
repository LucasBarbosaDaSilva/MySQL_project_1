# README

# Projeto All For One - Aprendizado em SQL e Banco de Dados com Node.js

## Contexto
Este é o repositório do projeto All For One, onde aprendi sobre SQL, banco de dados, queries, manipulação de dados e busca de dados. O objetivo deste projeto foi desenvolver uma aplicação utilizando Node.js e Docker Compose que me permitisse praticar e aprimorar minhas habilidades em SQL e manipulação de banco de dados. Durante o desenvolvimento deste projeto, explorei conceitos como modelagem de banco de dados, criação de tabelas, inserção de dados, atualização de registros, exclusão de dados e consultas utilizando diferentes tipos de queries.

## Tecnologias usadas

Desenvolvido usando: Node.js, SQL, Banco de Dados, Docker Compose.

## Pré-requisitos

Antes de executar o projeto, você precisará ter o Docker Compose instalado em sua máquina.

- [Instalação do Docker Compose](https://docs.docker.com/compose/install/)

## Executando o Projeto

Siga as etapas abaixo para executar o projeto com Node.js e Docker Compose:

1. Clone este repositório:
   git clone https://github.com/seu-usuario/nome-do-repositorio.git

2. Navegue até o diretório do projeto:
   cd nome-do-repositorio

3. Execute o seguinte comando para iniciar os serviços com o Docker Compose:
   docker-compose up
